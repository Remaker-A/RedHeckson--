# 余光 · 你的余光里，有个它一直在

> RED Hackathon 硬件赛道 · 深夜施工队

余光是一个住在帐篷里的数字伙伴。桌上的帐篷是它物理世界的居所，你来了它探出头，你走了它缩回去。它也生存在你的所有数字世界里——移动端、电脑端、甚至 Agent 的 Skill 里，通过软硬件静默感知你的状态，了解你、给你写信，和你一起成长。

你不用跟它聊天，它用自己的方式陪你。

---

## 项目结构

| 目录 | 说明 |
|------|------|
| `companion-agent/` | 陪伴 Agent 全栈工程 — FastAPI 后端 + Vue 3 前端（移动端 H5） |
| `accompany-mac/` | macOS 桌面端 — SwiftUI 原生应用，菜单栏常驻陪伴 |
| `CuteRobot/` | 硬件固件（主控）— Arduino，OLED 表情 + 舵机控制 + 串口通讯 |
| `CuteEyeRobot/` | 硬件固件（精简版）— Arduino Uno 适配，眨眼 + 表情动画 |
| `docs/` | 产品设计、叙事框架、视觉规范等文档 |
| `screenshots/` | 移动端界面截图 |

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 移动端 | Vue 3 + TypeScript，H5 全屏 PWA |
| 后端 | Python FastAPI + WebSocket，LLM 多模型适配 |
| AI/Agent | Soul 性格系统 + Memory 记忆 + 状态机 + Prompt 模板 |
| macOS 端 | SwiftUI，桌面上下文采集 |
| 硬件 | Arduino + ESP32，OLED 屏 + 舵机（俯仰/偏航）+ 串口协议 |
